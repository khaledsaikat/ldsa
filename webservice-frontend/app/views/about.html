Short Description

This is a Project of the University of Duisburg-Essen.
19 Students created a tool for analyzing Big-Data-Streams
from Social Media, especially Instagram. This tool can connect
to the Instagram Api and retrieve you all the Information
from the Instagram Endpoints. They will be stored in a Database
and you can analyze it even in realtime or for future analysis.
